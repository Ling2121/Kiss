# Kiss

## 项目结构

* assets  资源
  * image 图片
  * audio 音频
  * tileset 瓷砖集
  * font 字体
  * map 地图
* document  文档
* editor 编辑器
  * NoiseView 噪声视图（用于编辑噪声）
* library 第三方库
  * bump AABB碰撞库
  * HC 多边形碰撞库
  * camera 相机
  * dkjson JSON解析
* source 代码
  * core 框架核心
    * base 基础类库
    * collision 碰撞模块
    * resource 资源管理模块
    * scene 场景模块
    * constructor 对象构造器
    * animation 动画模块
  * entity 所有实体
  * component 所有组件
  * scene 场景
* test 测试代码
